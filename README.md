# 🔐 Password Strength Lab

A cybersecurity-focused web application that analyzes password security, identifies weaknesses, estimates crack resistance, and provides actionable recommendations to help users create stronger passwords.

---

## 🚀 Features

### 🔍 Real-Time Password Analysis

Instantly evaluates passwords as users type and provides a security score out of 100.

### 📊 Password Strength Scoring

Passwords are classified based on multiple security checks:

- Weak
- Fair
- Good
- Strong

### ⏳ Crack Time Estimation

Provides an estimated offline attack crack time based on password complexity.

### 🧪 Diagnostic Security Checks

The application verifies:

- Minimum 8 characters
- Recommended 12+ characters
- Lowercase letters
- Uppercase letters
- Numeric characters
- Special characters
- Common leaked password detection
- Repeated character detection
- Sequential pattern detection
- Weak password pattern detection

### 💡 Security Recommendations

Provides personalized suggestions for improving password security.

Example:

```text
✗ Add special characters
✗ Increase password length
✓ Contains uppercase letters
✓ Contains numbers
```

### 🔑 Strong Password Generator

Generates secure example passwords to help users create stronger credentials.

### 🔒 Privacy Focused

- Runs entirely in the browser
- No passwords are stored
- No passwords are transmitted
- No external processing

## 📈 Security Checks Performed

| Check | Description |
|---------|------------|
| Length Check | Verifies password length |
| Character Diversity | Checks lowercase, uppercase, numbers, symbols |
| Common Password Detection | Identifies frequently used passwords |
| Sequential Detection | Detects patterns like 123456 or abcdef |
| Repetition Detection | Detects repeated characters |
| Pattern Analysis | Identifies predictable password structures |
